I believe that everyone should have the basic idea about programming. You don't have to learn every single programming language on Earth and become a professional programmer, you just need a general idea of how programming works and how to generally read code. Almost all programming languages follow the same rules.

The advantages

Understanding and manipulating the source code

Since you understand the source code, you should also be able to make a few basic edits to it, example changing text of messages, disabling features you don't like or even modding if you're a bit more advanced.

Finding malicious code

If you get your hands on the source code of a program, you are able to check if that program does anything malicious (like sending HTTP requests to weird websites, reading sensitive files etc.).

You should also know how to spot obfuscated (unreadable) code. Hackers usually obfuscate their code to make reverse engineering their malware more difficult.
